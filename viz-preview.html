Pulse / Aliveness — How buildings show they're "live" and "loaded"

Current
Size pulse
Marker grows/shrinks. Frequency tied to load (idle = slow, peak = fast). What's on the map now.
ReferenceBaseline
Option A
Sonar ripples
Concentric rings expand outward periodically. Faster ripples = higher load. "Energy emanating" feel.
Best for "energetic" feelingDistinct
Option B
Halo brightness
Marker stays still; surrounding glow gently breathes in opacity. Subtle, less distracting.
Calmest optionSubtle
Option C
Heatmap halo
No motion. Halo color encodes load — yellow → orange → red. Static but information-dense.
Where is hot? At a glance.Info-dense
Option D
Trend arrows
Replaces "current load" cue with "where is it heading" — ▲ rising · → steady · ▼ falling.
Different info entirelyForecast-aware
Option E
Measurement flash
No continuous motion. Marker briefly flashes when new BMO data arrives (every 15min).
Means "data is live" not "load is high".

Category Encoding — How building type is shown

Current
Color-only dots
One color per category. Simple, but uses up "color budget" that could encode load too. Color-blind unfriendly.
ReferenceBaseline
Option A
Color + icon
Color stays, plus a tiny semantic icon — gear (eng), bed (resi), people (student life), book (academic), gov (admin).
Color-blind safe + semanticBest ROI
Option B
Shape per category
Same color (gold), shapes differ — circle/hex/square/triangle/diamond. Frees color for load/state.
Print-friendlyFrees color
Option C
Building footprints
Replace dots with actual building outlines from ArcGIS, filled by category. "Architectural map" feel.
Biggest visual changeArchitectural
Option D
Regional tint + neutral dots
Soft category-colored zones cover where each type lives. Markers all neutral — color freed for load/state.
Atmospheric, info-denseFrees color
Option E
Ring style per category
All same fill, ring varies — solid / dashed / double / dotted / thick. Quietly distinguishes without color.
Quiet, low-contrastSubtle

Promising Combinations — pulse × category together

Combo 1 — Gentle evolution
Sonar + icons
Keeps current vibe. Adds semantic clarity (icons) and more legible aliveness (sonar). Smallest change, low risk.
Lowest risk · most familiarEasy win
Combo 2 — Information-dense static
Heatmap halo + regional tint
Drops motion entirely. Color repurposed: halo = load intensity, region tint = category. One frame conveys everything.
Best for screenshots / slidesBig change
Combo 3 — Architectural
Footprints + trend arrows
Most dramatic shift. Buildings become real shapes, motion encodes direction (rising/steady/falling) instead of magnitude.
Biggest demo impact · most workAmbitious
